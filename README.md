# javascript-linked-list
Linked list implementation in Javascript

### Sample Usage
```javascript
let lList = new LinkedList();
lList.add(1);
lList.add(2);
lList.add(2);
lList.add(4);
lList.insert(0, 0);
lList.insert(5, 2);
console.log(lList.toList());
console.log(lList.pretty());
console.log(lList.remove(4));
console.log(lList.pretty());
```
